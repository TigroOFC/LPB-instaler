LPB - Ajuda

Como usar a LPB

1. Crie uma pasta chamada pasta visual_studio_code (certifique-se de ter o Visual Studio Code instalado).

2. Dentro da pasta, crie um projeto e coloque o arquivo dist.zip que você baixou.

3. Extraia o conteúdo do dist.zip no projeto.

4. Crie arquivos com extensão .lpb e escreva seu código LPB.

5. Para executar, abra o terminal no projeto e rode:

dist\LPB_interpretador.exe nome_do_arquivo.lpb

Comandos da LPB

variavel nome = valor → cria uma variável.

mostrar("texto") → exibe uma mensagem.

mostrar("texto:variavel") → exibe o texto seguido do valor da variável.

se condição faça ... fim → bloco condicional.

senao → usado junto com se para condição alternativa.

enquanto condição faça ... fim → loop enquanto a condição for verdadeira.

nome = expressão (esperar x) → atribui valor e pode esperar x segundos.

Dicas

- Use aspas para textos e strings.

- As variáveis podem ser números ou textos.

- Certifique-se de fechar blocos fim corretamente.

- Teste seu código em pequenos trechos antes de criar programas grandes.
